# 二、链表（指针操作）

## 核心思想

通过指针改变节点连接关系

---

## 例题1：反转链表（LeetCode 206）

**题干**：反转一个单链表。

```java
class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}

public ListNode reverseList(ListNode head) {
    ListNode prev = null;
    ListNode curr = head;
    while (curr != null) {
        ListNode nextTemp = curr.next;
        curr.next = prev;
        prev = curr;
        curr = nextTemp;
    }
    return prev;
}
```

**讲解**：

- 三指针：prev / curr / next
    
- 每次反转当前节点指向
    

---

## 例题2：环形链表（LeetCode 141）

**题干**：判断链表是否有环。

```java
public boolean hasCycle(ListNode head) {
    ListNode slow = head, fast = head;
    while (fast != null && fast.next != null) {
        slow = slow.next;
        fast = fast.next.next;
        if (slow == fast) return true;
    }
    return false;
}
```

**讲解**：

- 快慢指针：快2步，慢1步
    
- 有环必相遇

# 三、数组 + 双指针 / 滑动窗口

## 核心思想

通过指针移动降低时间复杂度

---

## 例题1：三数之和（LeetCode 15）

**题干**：找出所有和为0的不重复三元组。

```java
import java.util.*;

public List<List<Integer>> threeSum(int[] nums) {
    Arrays.sort(nums);
    List<List<Integer>> res = new ArrayList<>();
    int n = nums.length;

    for (int i = 0; i < n; i++) {
        if (i > 0 && nums[i] == nums[i - 1]) continue;
        int left = i + 1, right = n - 1;

        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];
            if (sum == 0) {
                res.add(Arrays.asList(nums[i], nums[left], nums[right]));
                left++;
                right--;
                while (left < right && nums[left] == nums[left - 1]) left++;
                while (left < right && nums[right] == nums[right + 1]) right--;
            } else if (sum < 0) {
                left++;
            } else {
                right--;
            }
        }
    }
    return res;
}
```

**讲解**：

- 排序 + 双指针
    
- 固定一个数，转为两数问题
    

---

## 例题2：无重复字符的最长子串（LeetCode 3）

**题干**：求最长无重复子串长度。

```java
import java.util.*;

public int lengthOfLongestSubstring(String s) {
    Set<Character> set = new HashSet<>();
    int left = 0, maxLen = 0;

    for (int right = 0; right < s.length(); right++) {
        while (set.contains(s.charAt(right))) {
            set.remove(s.charAt(left));
            left++;
        }
        set.add(s.charAt(right));
        maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
}
```

**讲解**：

- 滑动窗口维护区间
    
- 遇重复收缩左边界

# 四、DFS / BFS（遍历）

## 核心思想

遍历树或图的所有节点

---

## 例题1：二叉树层序遍历（LeetCode 102）

**题干**：返回二叉树的层序遍历。

```java
import java.util.*;

class TreeNode {
    int val;
    TreeNode left, right;
    TreeNode(int x) { val = x; }
}

public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    if (root == null) return res;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int size = queue.size();
        List<Integer> level = new ArrayList<>();

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);

            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        res.add(level);
    }
    return res;
}
```

**讲解**：

- BFS使用队列
    
- 按层遍历
    

---

## 例题2：岛屿数量（LeetCode 200）

**题干**：计算岛屿数量。

```java
public int numIslands(char[][] grid) {
    int rows = grid.length, cols = grid[0].length;
    int count = 0;

    for (int r = 0; r < rows; r++) {
        for (int c = 0; c < cols; c++) {
            if (grid[r][c] == '1') {
                dfs(grid, r, c);
                count++;
            }
        }
    }
    return count;
}

private void dfs(char[][] grid, int r, int c) {
    int rows = grid.length, cols = grid[0].length;
    if (r < 0 || r >= rows || c < 0 || c >= cols || grid[r][c] == '0') return;

    grid[r][c] = '0';
    dfs(grid, r + 1, c);
    dfs(grid, r - 1, c);
    dfs(grid, r, c + 1);
    dfs(grid, r, c - 1);
}
```

**讲解**：

- DFS遍历连通块
    
- 访问后标记

# 五、回溯（搜索）

## 核心思想

DFS + 撤销选择

---

## 例题1：全排列（LeetCode 46）

**题干**：返回所有排列。

```java
import java.util.*;

public List<List<Integer>> permute(int[] nums) {
    List<List<Integer>> res = new ArrayList<>();
    boolean[] used = new boolean[nums.length];
    backtrack(nums, new ArrayList<>(), used, res);
    return res;
}

private void backtrack(int[] nums, List<Integer> path, boolean[] used, List<List<Integer>> res) {
    if (path.size() == nums.length) {
        res.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) continue;

        path.add(nums[i]);
        used[i] = true;

        backtrack(nums, path, used, res);

        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

**讲解**：

- 选择 → 递归 → 撤销
    

---

## 例题2：组合总和（LeetCode 39）

**题干**：找出和为目标的组合。

```java
import java.util.*;

public List<List<Integer>> combinationSum(int[] candidates, int target) {
    List<List<Integer>> res = new ArrayList<>();
    Arrays.sort(candidates);
    backtrack(candidates, target, 0, new ArrayList<>(), res);
    return res;
}

private void backtrack(int[] candidates, int target, int start, List<Integer> path, List<List<Integer>> res) {
    if (target == 0) {
        res.add(new ArrayList<>(path));
        return;
    }
    if (target < 0) return;

    for (int i = start; i < candidates.length; i++) {
        path.add(candidates[i]);
        backtrack(candidates, target - candidates[i], i, path, res);
        path.remove(path.size() - 1);
    }
}
```

**讲解**：

- start避免重复
    
- 剪枝减少搜索

# 六、动态规划（DP）

## 核心思想

利用子问题结果推导当前问题

---

## 例题1：爬楼梯（LeetCode 70）

**题干**：每次1或2步，求总方法数。

```java
public int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) {
        int temp = a + b;
        a = b;
        b = temp;
    }
    return b;
}
```

**讲解**：

- 状态转移：dp[i] = dp[i-1] + dp[i-2]
    

---

## 例题2：最长公共子序列（LeetCode 1143）

**题干**：求两个字符串的LCS长度。

```java
public int longestCommonSubsequence(String text1, String text2) {
    int m = text1.length(), n = text2.length();
    int[][] dp = new int[m + 1][n + 1];

    for (int i = 1; i <= m; i++) {
        for (int j = 1; j <= n; j++) {
            if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                dp[i][j] = dp[i - 1][j - 1] + 1;
            } else {
                dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
            }
        }
    }
    return dp[m][n];
}
```

**讲解**：

- 相等：来自左上
    
- 不等：取最大值
